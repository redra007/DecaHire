## Image Optimization Tips

Before uploading to GitHub:
- Use JPG for photos (smaller file size)
- Use PNG for logos (transparent background)
- Compress images: https://tinypng.com or https://squoosh.app
- Target: Keep each image under 200KB
- Total images folder: Under 5MB ideally

---

## Testing After Changes

1. Save index.html
2. Upload to GitHub (or push changes)
3. Wait 1-2 minutes
4. Refresh your website
5. Check browser console (F12) for any image loading errors
