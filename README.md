# Images Folder

## Upload Your Logo Here

1. **Logo Requirements:**
   - Filename: `logo.png` (must be exactly this name)
   - Format: PNG with transparent background (recommended) or JPG
   - Size: 200-400px width recommended
   - File size: Under 100KB

2. **Your Other Images:**
   - Upload all replacement images here
   - Name them according to the IMAGE_REPLACEMENT_GUIDE.txt
   - Recommended formats: JPG (for photos), PNG (for graphics/logos)

## Image List Needed:

## Upload Methods:

### Via GitHub Web:
1. Click on "images" folder in your repository
2. Click "Add file" → "Upload files"
3. Drag and drop your images
4. Commit changes

### Via Git Command Line:
```bash
cd decahire-website/images
# Copy your images here
git add .
git commit -m "Add website images"
git push
```

## Tips:
- Optimize images before upload: https://tinypng.com
- Keep individual files under 200KB
- Total folder should be under 5MB
- Use descriptive filenames
- Test on website after uploading
